# Survey
This is an app that functions like a cash register.  There are some errors I found when installing on my home computer that will need fixing.
## Setup Instructions:
### Gems Used:
```
sinatra
sinatra-contrib
sinatra-activerecord
rake
pg
pry
rspec
shoulda-matchers
```
### Install Bundler:
```
$ gem install bundler
```
### Run Bundler:
```
$ bundle
```
### Start the database:
```
$ postgres
```
### Create databases:
```
$ rake db:create
$ rake db:migrate
$ rake db:test:prepare
```
